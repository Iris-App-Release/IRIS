# IRIS v0.1 Alpha Release

IRIS turns your display into a window into a 3D world. As you move your head, the scene shifts perspective in real time, creating the illusion of depth behind the screen.

## What's New in v0.1

* **Multiple Worlds** — Explore Earth, Gem, The Watcher, and more
* **Improved World Selector** — Choose your favorite world at launch
* **Better Performance** — Optimized rendering and tracking
* **Enhanced Parallax** — Smoother, more immersive head-tracking illusion
* **Stability Improvements** — Bug fixes and tracking refinements

## Features

Real-time head tracking • Multiple 3D environments • Desktop wallpaper mode • Dynamic parallax perspective • World-based modular architecture

## ⚠️ Important: Installation Instructions

1. **Download** the DMG from Drive
2. **Open DMG** and drag Iris.app to Applications
3. **First launch:** macOS will block the app (unsigned). Go to **Settings → Privacy & Security** and scroll down to find "Iris" 
4. **Click "Open Anyway"** to allow the app to run
5. Grant camera permission when prompted

## System Requirements

* Apple Silicon only (M1, M2, M3, M4)
* Intel Macs not supported
* macOS only

## Alpha Status

This is **v0.1** — expect bugs, rough UI, and occasional glitches. This is a technology preview to test core experience and gather feedback.

## Feedback Welcome

Report bugs, tracking issues, performance problems, and feature ideas. Your feedback shapes what IRIS becomes next.
